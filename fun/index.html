Fun is very important!

  • Whenever you meet a new group of people in some organized setting, there's always bound to be that one horribly uncreative organizer (don't be that person!) whose default faux-icebreaker question is "What is a fun fact about you?" For a long time, I just made up fun facts about myself. I've always wanted to be able to have a default fun fact to answer this default question. Now I have one: I'm almost left-handed!
  • Did you know that "default" isn't an adjective? In fact, there's so much wrong with this word, and other words, that I have a whole section of this website dedicated to mini-rants over the many, many problems with words! (Just kidding—not yet. But it's coming soon!)
